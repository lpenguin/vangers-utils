# Vangers utils

## Dependencies
* Python >= 3.4

## Installation
```bash
python3 setup.py install
vangers-utils -h
```

```
usage: vangers-utils <command> [<args>...]

The most commonly used commands are:
   scb        Decode/encode scb
   bmp        Decode/encode bmp
   xbm        Decode/encode xbm

```
